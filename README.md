# 📊 BI Dashboard для анализа закупок

[![License: MIT](https://img.shields.io/badge/License-MIT-blue.svg)](https://opensource.org/licenses/MIT)
[![JavaScript](https://img.shields.io/badge/JavaScript-ES6+-F7DF1E?logo=javascript&logoColor=black)](https://developer.mozilla.org/en-US/docs/Web/JavaScript)
[![Chart.js](https://img.shields.io/badge/Chart.js-3.9-FF6384?logo=chartdotjs&logoColor=white)](https://www.chartjs.org/)

> **Интерактивная приборная панель для аналитики закупок, поставщиков и бюджетного планирования**

Полнофункциональный инструмент для менеджеров по закупкам и финансовых аналитиков. Позволяет отслеживать KPI поставщиков, контролировать исполнение бюджета, формировать рекомендации по заказам и строить прогнозы.

## ✨ Основные возможности

| Модуль | Функционал |
|--------|-----------|
| **Поставщики** | Анализ цен, сроков поставок, рейтингов и просрочек |
| **Бюджет** | План-факт анализ, визуализация списания бюджета, распределение по филиалам |
| **Рекомендации** | Автоматический расчёт потребности в закупках с учётом прогноза |
| **Прогнозы** | Трендовые модели, анализ "Что если", ROI поставщиков |

## 🧠 Ключевые сценарии использования

### 1️⃣ Аналитика поставщиков
- **Фильтрация:** по категории, филиалу, периоду (месяц/квартал/год/произвольный)
- **Графики:** тренд цен, цена vs срок поставки, рейтинг поставщиков
- **Интерактив:** редактирование/удаление/добавление поставщиков, детальная карточка
- **KPI:** средний рейтинг, срок поставки, процент просрочек

### 2️⃣ Бюджет закупок
- **План-факт:** таблица + столбчатая диаграмма по месяцам
- **Гаджеты:** спидометр выполнения бюджета, круговая диаграмма по филиалам
- **Анализ "Что если":** изменение плановых значений с пересчётом остатка
- **Сравнение периодов:** быстрая оценка динамики

### 3️⃣ Рекомендации по закупкам
- **Расчёт:** (Прогноз × Коэффициент) - (Остаток + В пути)
- **Редактирование:** ручная корректировка заказа
- **Передача в 1С:** экспорт предложений в JSON или сохранение в localStorage

### 4️⃣ Прогнозирование
- **Бюджетный прогноз:** линейная экстраполяция на 3/6/12 месяцев
- **Прогноз просрочек:** тренд ухудшения показателей
- **ROI поставщиков:** автоматический расчёт на основе рейтинга и цены

## ⚙️ Расширенные возможности

### 🌓 Тёмная тема
Автоматическое сохранение выбранной темы в localStorage.

### 💾 Сохранение фильтров
Фильтры по поставщикам сохраняются между сессиями.

### 🔄 Автообновление данных
- Интервал: 120 секунд
- Визуальный обратный отсчёт
- Возможность временной остановки

|-----------|----------|

## 🛠️ Администрирование

### Локальное хранение данных
Данные сохраняются в localStorage:
- `columnSettings` — настройка видимости столбцов
- `savedFilters` — сохранённые фильтры
- `theme` — выбранная тема
- `1c_transfers` — история передач в 1С

### Сброс данных
```javascript
// Очистить все локальные настройки
localStorage.clear();
```
